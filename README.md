# Karl Cometa Digital Marketing Portfolio Website

A static, Netlify/Vercel-ready personal portfolio website for Karl Cometa.

## Files

- `index.html` — main website
- `style.css` — styling and responsive layout
- `script.js` — animations, mobile menu, gallery filters, lightbox
- `assets/img/` — optimized portfolio images
- `assets/video/` — compressed video samples
- `assets/docs/` — CV and PDF portfolio
- `netlify.toml` — Netlify config
- `vercel.json` — Vercel config

## Deploy on Netlify

1. Extract the ZIP.
2. Go to Netlify.
3. Choose **Add new site → Deploy manually**.
4. Drag this whole folder into Netlify.
5. Netlify will generate your live URL.

## Deploy on Vercel

1. Extract the ZIP.
2. Upload to GitHub.
3. Import the repository in Vercel.
4. Framework preset: **Other**.
5. Deploy.

## Recommended domain

- Primary: `karlcometamarketing.com`
- Alternative: `karlcometa.com`

## Privacy reminder

Before publishing publicly, check every screenshot and remove or blur:
- Customer names
- Phone numbers
- Email addresses
- Bank/payment details
- Account balances
- Internal client-sensitive data
- Private Messenger conversations

This package avoids using the most sensitive tracker screenshots as hero proof, but you should still review all assets before going live.
